# Copy-Link-as-Hyperlink
# Privacy Policy
## Copy Link as Hyperlink
Last updated: 2026-03-13

Copy Link as Hyperlink does not collect, store, transmit, or share any personal data.

## Information Processed
The extension temporarily processes the following information only when explicitly triggered by the user:
- The URL of the current page
- The URL of a right-clicked link
- The visible text of a right-clicked link
This information is processed locally within the user's browser to generate a formatted hyperlink (HTML or Markdown).

## Data Collection and Storage
- No data is transmitted to external servers.
- No personal information is collected.
- No browsing history is stored.
- No analytics or tracking tools are used.
- No data is saved outside the user’s browser session.

## Third-Party Sharing
This extension does not share any data with third parties.

## Remote Code
The extension does not load or execute remote code. All functionality is contained within the extension package.
